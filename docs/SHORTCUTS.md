# Shortcuts

Customize these in Settings > Shortcuts.
| New Image              | `Ctrl+N`       |
|------------------------|----------------|
| Open File (in Project) | `Ctrl+O`       |
| Open File (New Tab)    | `Ctrl+Shift+O` |
| Save Project           | `Ctrl+S`       |
| Load Project           | `Ctrl+L`       |
| Export Image           | `Ctrl+E`       |
| Toggle Fullscreen      | `F11`          |
